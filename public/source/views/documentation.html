Get started

Hi, Be dummy is free app, designed to be easy to prototype mock data for your frontend or even backend components.

So first thing to do is actually to think what data you want to generate.

Next thing to do is to go to Be dummy's Models and start creating your prototypes.

Finally after creating your first model, you would like to see and do all sorts of stuff with your data right?

Be dummy app comes with basic type of Postman like testing application. In case you don't wanna install 3th party software, you can go to Quick postman section and use are basic like Postman app.

And thats all you basically need to know to start your journey in creating mock data.

What is Model

Basically model is object or row in your database, that represents some kind of record.

Example: some simple product

Your simple product should to look something like this

product_id name description price quantity
1 Basic product This is just a basic product $10 1000

Now we can "take" this data and convert it into model.

First create new Model

We should name it Simple product or however you need or want.

Then we can just drag and drop types from our right side to middle.

Name them as we shown above, and take types that are similar or same as above represented ones.

Save your Model and thats all. Congrats, now you can use it however you want.

Code examples

Code below is just there to show you how to call it from your favorite framework or library.

                        
                            //
                            // jQuery
                            //
                            $.ajax({
                                url: '{{websiteRoute}}/api/example',
                                success: function(response){
                                  console.log(response)
                                }
                            });
                        

                        
                            //
                            // React
                            //
                            axios.get('{{websiteRoute}}/api/example')
                                .then(function (response) {
                                    console.log(response)
                                })
                        

                        
                            //
                            // vue js
                            //
                            this.$http.get('{{websiteRoute}}/api/example')
                                .then((response) {
                                    console.log(response)
                                })
                        

                        
                            //

                            //
                            $http.get('{{websiteRoute}}/api/example')
                                .then((response) {

                                })
